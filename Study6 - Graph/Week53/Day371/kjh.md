# 회고

## 스터디를 하게 된 계기
```
졸업할 즈음, 위기감이 들었다.
학과에서 우수한 성적을 받은 것 말고는 딱히 이뤄낸 게 없는거다.

노마드코더 강의를 결제해 들어보고, 프로그래밍 책을 사서 혼자 막 공부하려 해봤다.
하지만 강의는 듣다 말았고, 책도 30%정도 보다가 덮었다.

나는 게을렀다. 혼자 공부를 지속하는건 어려웠다.

그래서 '알고리즘 스터디'를 동기들에게 제안했다.
'매일 코딩을 할 수 밖에 없는 환경'을 만들기 위해.
```

